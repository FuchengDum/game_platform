# 🧱 打砖块游戏 (Breakout)

## 📖 游戏简介

打砖块是一款经典的街机游戏，玩家通过控制挡板反弹小球来击碎屏幕上的砖块。该游戏结合了精确的控制、策略规划和反应速度，提供令人上瘾的游戏体验。

## 🎮 游戏规则

### 基本玩法
- **控制挡板**：使用左右箭头键或A/D键控制挡板移动
- **反弹小球**：利用挡板反弹小球，击碎上方的砖块
- **消除砖块**：击碎所有砖块即可进入下一关
- **保持小球**：不要让小球掉落到底部

### 游戏特性
- **多彩砖块**：不同颜色的砖块有不同的分值
- **特殊砖块**：需要多次击打才能消除的坚固砖块
- **关卡进阶**：随着关卡提升，难度逐渐增加
- **连击奖励**：连续击碎砖块获得额外分数

## 🎯 操作说明

| 控制键 | 功能 | 说明 |
|--------|------|------|
| ← / A | 左移挡板 | 向左移动挡板 |
| → / D | 右移挡板 | 向右移动挡板 |

## 💡 游戏技巧

### 新手技巧
1. **掌握挡板控制**：练习精准的挡板移动
2. **预判小球轨迹**：观察小球的角度，提前准备
3. **优先消除角落**：从边缘开始，逐渐向中间推进
4. **保持节奏**：不要急于求成，保持稳定的节奏

### 进阶技巧
1. **角度控制**：利用挡板边缘改变小球反弹角度
2. **连击策略**：寻找可以连续击碎多个砖块的角度
3. **砖块优先级**：优先清除坚固砖块和特殊砖块
4. **挡板位置**：将挡板保持在屏幕中间区域，便于左右移动

## ⚙️ 游戏配置

### 难度设置
- **简单模式**：挡板速度较快，小球速度较慢
- **普通模式**：平衡的游戏速度和挑战性
- **困难模式**：挡板速度较慢，小球速度较快

### 视觉设置
- **粒子效果**：开启/关闭砖块破碎效果
- **音效设置**：调节游戏音效音量
- **主题颜色**：选择不同的游戏主题

## 🔧 技术说明

### 游戏架构
- **引擎**：基于 Phaser 3.x 游戏引擎
- **物理系统**：内置物理引擎处理碰撞检测
- **渲染系统**：Canvas 2D 渲染，支持硬件加速
- **输入系统**：响应式键盘控制，支持连续输入

### 性能优化
- **对象池**：复用粒子效果对象，减少内存分配
- **碰撞优化**：使用空间分区算法优化碰撞检测
- **帧率控制**：稳定的60FPS游戏体验
- **资源管理**：智能加载和缓存游戏资源

## 🐛 常见问题

### Q: 游戏卡顿怎么办？
A: 确保浏览器支持硬件加速，关闭其他占用GPU的程序。

### Q: 控制不灵敏？
A: 检查键盘是否有其他按键冲突，尝试刷新页面。

### Q: 小球卡住不动？
A: 这是极少数情况，按R键重新开始当前关卡。

### Q: 游戏分数没有保存？
A: 本地存储需要浏览器支持，确保没有禁用Cookies。
